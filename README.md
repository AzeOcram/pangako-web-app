# pangako-web-app
- Pangako is a simple tracking website for politicians running for positions during elections. Our goal is to fight against misinformation and provide accurate data by creating a central source of information for all activities and projects that are involved by said politicians. All of the information we provide is properly referenced, all from credible government sources accessible by anyone. 

- Viewers are also able to contribute to this project through a feedback submission bin on our website. By sending us the politician concerned, a short description and most importantly, the source url for the information,  you can contribute towards spreading awareness and accurate data for everyone to see. Submissions are carefully read by admins in order to prevent any untrustable sources or troll submissions from making it into the database.
